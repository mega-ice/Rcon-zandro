# Rcon-zandro

## A basic Zandronum v.3+ RCon console terminal.

The Zandronum RCon protocol uses its own (SKULLTAG) Huffman frequencies, which are implemented in huffman.py.

The program uses the IP address and port as arguments; the server password is stored directly in the rcon_client.py file.

Usage: python3 rcon_client.py 127.0.0.1 7777
